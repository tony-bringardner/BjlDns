 DNS Project:
 
 The DNS library provides everything you need for a DNS server and
 a DNS client. 
 
 Requires Java 1.5 or greater
 Dependencies:  
 	Compile: 
 			BjlCore
 			BjlIo
 			
 	Runtime: 
 		Required:
 			BjlCore
 			BjlIo 		
 		Optional:
 			log4j
